### **2)Поиск имен собственных**
Чтобы найти все имена собственные, оканчивавшиеся на -слав, я пользовалась регулярным вырожением:
[А-Я][а-яѣ]+слав[а-яѣ]+
Всего упоминаний: 564

![](https://github.com/VictoriaBelasheva/hw9/blob/master/2.png?raw=true)

### **3)Поиск упоминаний Новгорода**
